> 但是如果我们传入非字符串类型的数据的时候，这个函数将发生错误，在5.3之前的php中，显示了报错的警告信息后，将return 0 ! 也就是虽然报了错，但却判定其相等了。

> 解这道题，就只要传入一个非字符串类型的变量即可，一般情况下，我们我们传数组
